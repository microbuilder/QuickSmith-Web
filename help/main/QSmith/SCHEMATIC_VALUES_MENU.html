Schematic: Assign Values Menu

Frequency

This command lets user to change the current frequency in MHz.

 

Velocity Factor

This command prompts user to change the velocity factor. The default value is 1.

The velocity factor is the same for all transmission lines.

 

Characteristic Impedance

This command lets user to change the characteristic impedance Zo. The default is 50 Ohms.

Transmission lines can have different Zo values.

 

Transmission Line Design Frequency

This command lets user to change Transmission line design frequency which can be different from the analysis frequency. The Transmission line design frequency is also known as reference frequency.

 

Element Q

This command lets user to change the Q value of each element.

Since all elements have finite Q values, the Q values are included in the analysis to get more practical results.

The default Q value is 1000000.

 

Set To Defaults

This command sets the Frequency, Velocity Factor, Zo, Transmission line frequency and Element Qs to their default values.
