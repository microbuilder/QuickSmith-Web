Smith Chart: Gamma File

A "Gamma File" is an ASCII file. This file can be created/edited using QuickSmith editor or by exporting data out of QuickSmith. The following are the characteristics of "Gamma File".

The file name should have the "GAM" extension.

The first line in the file should be "QuickSmith GAM".

The second line is the characteristic impedance ,Zo, which is usually 50 ohms.

The rest of the file contains the values of frequency in MHz, reflection coefficient magnitude and angle each separated by a space and finally followed by a carriage return.,
