Smith Chart: Introduction

Introduction

Smith Chart was invented by Phillip H. Smith and is extensively used for radio/microwave frequency circuit design. QuickSmith is a Smith Chart program designed to aid in Smith chart plotting and calculations. This program is also capable of determining the insertion loss in a network or wave guide.

The goal of this program is to make it as user friendly as possible. This program has been designed to be used not just by professionals but also by students and hobbyist. It has been intended that this program will make graphical based design more accurate, easier, faster and fun.

Some of the features offered by this program are listed below

 

:Frequency/component sweep

:Load impedance interpolation/extrapolation for multiple frequency terminations

:Q factors taken into account for network components

:S-Parameter based amplifier design/analysis

:Insertion loss or S21 graphs

:Transmission line parameters calculations

:Constant VSWR and Q circles
