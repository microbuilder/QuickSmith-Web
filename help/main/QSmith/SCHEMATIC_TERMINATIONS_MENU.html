Schematic: Terminations Menu

Impedance

This command displays load termination in terms of complex impedance.

 

Gamma

This command displays load termination in terms of Gamma (reflection coefficient).

 

Single

This command makes the current termination constant for all frequencies.

 

Multiple

This command lets the user to edit and link multiple terminations, this mode is indicated by the letter "M" placed next to the load in the schematic.

A narrow band application typically uses a single termination however this is not of much value for a broad band application. In practical situations the load varies with frequency and more so if the frequency range is large, a very good example of this will be a dipole antenna. QuickSmith overcomes this problem by varying the load with the frequency. As many as 1000 load points can be entered and linked to the design, these points are entered in ascending order with respect to their corresponding frequency and are similar to the "Gamma file" format.

QuickSmith uses polynominal iterpolation and extrapolation algorithms to determine load termination at any given frequency.
