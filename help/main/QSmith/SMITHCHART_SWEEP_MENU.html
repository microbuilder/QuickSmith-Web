Smith Chart: Sweep Menu

Generate Sweep

This command draws the user defined sweep trace on the chart .The sweep details can be set through the Sweep Menu in the Schematic window. Sweep can vary either frequency or component value, the default swept element is frequency.

 

Clear Trace

This command clears the sweep trace on the chart.

 

Sweep Trace Color

This command opens the color-palette window enabling the user to choose trace color for sweep display on the chart.
