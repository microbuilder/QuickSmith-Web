Schematic: File Menu

New

This Command initializes the schematic editor.The existing schematic is erased and replaced with

a short on the series slot and a open on the parallel slots.

 

Open Schematic

This command opens a previously saved schematic file and loads it in the ladder network.

 

Save Schematic As

This command saves the circuit on to a schematic file.

 

Print Schematic

Use this command to print the schematic.

 

Edit Schematic

Use this command to browse/edit a previously saved schematic file.

 

Exit

This command ends QuickSmith.
