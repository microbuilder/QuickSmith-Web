Amplifier Design/Analysis: Circles Menu

Stability circles

This command draws "Stability Circles" on the Smith Chart.

"Stability Circles" item is enabled after S-parameters have been entered.

"Stability Circles" can be turned on or off by clicking the "Stability Circles" item.

 

Operating Gain Circles

"Operating Gain Circles" Item is enabled after S-parameters have been entered.

This command draws "Available Gain Circles" on the load plane of the Smith Chart.

The user is prompted to enter the initial "Gain [Ga]" which can be incremented/decremented by its "Step Size" using the spin control located next to "Gain" text box on the bottom of the screen.

"Operating Gain Circles" can be turned on or off by clicking the "Operating Gain Circles" item.

 

Available Gain Circles

"Available Gain Circles" item is enabled after S-parameters have been entered.

This command draws "Available Gain Circles" on the source plane of the Smith Chart.

The user is prompted to enter the initial "Gain [Ga]" which can be incremented/decremented by its "Step Size" using the spin control located next to "Gain" text box on the bottom of the screen.

"Available Gain Circles" can be turned on or off by clicking the "Available Gain Circles" item.

 

Noise Circles

"Noise Circles" item is enabled after noise parameters have been entered.

This command draws constant "Noise Circles" on the source Pane of the Smith Chart.

The "Noise Figure [Fi]" can be incremented/decremented by its "Step Size" using the spin control located next to "Fi" text box on the bottom of the screen.

"Noise Circles" can be turned on or off by clicking the "Noise Circles" item.
