Amplifier Design/Analysis: Enter Parameters Menu

S-Parameters

This command opens a box for the user to enter bilateral S-Parameter information. Once the data has been entered the user can determine the stability information by clicking on the stability button. QuickSmith then calculates the K and delta values and displays the stability message. It also enables the stability and gain circles to be plotted.

"Stability Circle" can be turned on or off using the "Stability Circle" item in Circles Menu

 

Noise Parameters

This command opens a box for the user to enter Noise-Parameter information. Once the data has been entered the user can click "OK" to plot the "Noise Circle".

"Noise Circle" can be turned on or off using the Noise Circle Item in Circles Menu

 

Characteristic Impedance

This command prompts the user to enter characteristic impedance of the charts.
